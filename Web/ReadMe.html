What is Dove?

Dove is a Docking Model Evaluation method based on 3D Deep Convolutional Neural Networks

Dove is an improved evaluation method for docking model, which is based on cross-fold training and validation on around 2 million examples. Compared to previous method, it can work better on different benchmarks. Meanwhile, for most incorrect models, at least one of Dove method will give a very low probability(<0.01).

Dove protocol

Dove protocol consists of mainly four steps:

(1) Calculate modified GOAP and ITScore to get each atom's score;

(2) Extract the interface atom type, location and energy score to form the input for deep learning model;

(3) Use the deep learning trained model to predict the probability
